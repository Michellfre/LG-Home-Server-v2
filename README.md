# LG Home Server v3.1

Servidor doméstico para Android usando Termux + Nginx.

## Recursos

- Dashboard web
- Status em JSON
- IP local
- Nginx ativo/parado
- Espaço livre
- Contagem de arquivos das câmeras
- Contagem de backups
- Scripts de instalação, atualização e limpeza

## Instalação

```bash
bash install.sh
nginx -s reload
```

## Atualização

```bash
git pull
bash update.sh
```
