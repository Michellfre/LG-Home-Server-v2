# v13.2 Discovery Engine

Incluído:
- Endpoint `/api/discovery/scan`.
- Compatibilidade com `/api/connect/network/scan`.
- ARP table via `ip neigh`.
- Ping e varredura de portas.
- Detecção de HTTP, RTSP, MQTT, SMB, Home Assistant, Chromecast, impressora e Open Home API.
- Classificação com confiança.
- Merge automático dos dispositivos descobertos no Open Home Connect.
- Jarvis: "descobrir rede".
